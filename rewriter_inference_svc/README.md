# Rewriter Inference Service

Production-grade inference microservice for the **phi-3-mini** prompt rewriter model, serving as the actor in an Actor–Critic PPO training pipeline.

## Overview

The service accepts clinical notes and returns:
- **rewritten_prompt** — rewritten prompt generated by phi-3-mini
- **log_prob_old** — log probability of the generated sequence under the current policy
- **value_estimate** — scalar value predicted by the value head

## Setup

```bash
cd rewriter_inference_svc
pip install -r requirements.txt
```

## Running the Service

```bash
uvicorn app:app --host 0.0.0.0 --port 8000
```

## API

### POST /rewrite_prompt

**Request:**
```json
{
  "clinical_note": "Patient presents with chest pain and shortness of breath."
}
```

**Response:**
```json
{
  "rewritten_prompt": "...",
  "log_prob_old": -10.23,
  "value_estimate": 0.41
}
```

## Checkpoint Loading

The service automatically selects the checkpoint:
1. If `rl_checkpoints/` contains any checkpoint → loads the **latest** (by modification time).
2. Otherwise → loads from `sft_checkpoints/`.
